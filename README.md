Polityka Prywatności aplikacji „Mechan Kicia"

Niniejsza Polityka Prywatności określa zasady ochrony danych
użytkowników korzystających z aplikacji mobilnej „Mechan Kicia"
(„Aplikacja").

Aplikacja została stworzona przez uczniów Zespołu Szkół
Politechnicznych w Głogowie z okazji 80-lecia szkoły.

§1. Administrator danych
Administratorem aplikacji jest zespół uczniów odpowiedzialnych
za rozwój aplikacji „Mechan Kicia" w ramach projektu szkolnego
Zespołu Szkół Politechnicznych w Głogowie.

§2. Zakres zbieranych danych
Aplikacja „Mechan Kicia" nie zbiera, nie przechowuje ani nie
przetwarza żadnych danych osobowych użytkowników.

Aplikacja nie wymaga:
- logowania ani rejestracji
- dostępu do internetu
- dostępu do lokalizacji, kontaktów ani plików urządzenia
- żadnych uprawnień systemowych

§3. Cele przetwarzania danych
Ponieważ aplikacja nie zbiera żadnych danych, nie ma miejsca
żadne przetwarzanie danych użytkowników.

§4. Udostępnianie danych
Aplikacja nie udostępnia żadnych danych podmiotom trzecim,
ponieważ żadne dane nie są zbierane.

§5. Bezpieczeństwo
Aplikacja działa w całości lokalnie na urządzeniu użytkownika.
Żadne dane nie są przesyłane przez internet.

§6. Prawa użytkownika
Ponieważ aplikacja nie zbiera żadnych danych osobowych,
nie ma danych do wglądu, sprostowania ani usunięcia.

§7. Pliki cookies i analityka
Aplikacja nie wykorzystuje plików cookies, narzędzi
analitycznych ani żadnych technologii śledzących.

§8. Usługi zewnętrzne
Aplikacja nie korzysta z żadnych usług zewnętrznych ani
podmiotów trzecich.

§9. Zmiany Polityki Prywatności
Administrator zastrzega sobie prawo do wprowadzania zmian
w niniejszej Polityce Prywatności. Aktualna wersja będzie
zawsze dostępna na stronie GitHub projektu.
